What is docker?

"Docker is to deployment as Git is to development"

What is docker?

  • Open platform for building and sharing disk images
  • Lightweight VMs
  • Docker Hub Registry

How does it work?

  • Linux Containers (LXC) - fully isolated processes
  • Btrfs (B-tree file system) - snapshots

  • lightweight Linux distribution (Tiny Core Linux)

Dockerfiles

  • Recipe to build a container ("Makefile")
  • Based on a image (FROM)
  • Additional commands (RUN)

Example


FROM ubuntu:14.04

RUN apt-get update && apt-get install -y nginx

CMD nginx -g "daemon off;"

EXPOSE 80

fig.yml


web:
  build: .
  command: python app.py
  ports:
   - "5000:5000"
  volumes:
   - .:/app
  links:
   - redis

  image: redis
